## 🚀 Getting Started  

### 📥 Clone the Repository  

```bash
git clone https://github.com/satyajit1106/GenAi.git
cd GenAi
```

### 🔧 Install Dependencies  

Ensure you have **Python** installed, then run:  

```bash
pip install -r requirements.txt
```

### 🛠 Run Your First AI Model  

```bash
python main.py
```
